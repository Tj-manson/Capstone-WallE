# Capstone-WallE
## Features
- 6 randomized autonomous behaviours: head rotation, head 
  nod, mirrored arm waves, door opening and closing, forward 
  motion through treads, and look-around sequence
- Boot-up animation with buzzer tones on startup
- Ultrasonic sensor obstacle detection with reaction: look around and turn

## Hardware
- Arduino MEGA
- 5x servo motors (head rotate, head tilt, arms, door)
- DC motors with L298N motor driver
- HC-SR04 ultrasonic sensor


## Language
C / Arduino IDE
